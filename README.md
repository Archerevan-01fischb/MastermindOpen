## Quick taste

```bash
git clone https://github.com/{YOUR_GITHUB}/OpenMastermind.git
cd OpenMastermind
python init.py
```

Then open Claude Code in your project root and say:
*"Mastermind waking up for the first time."*

5 minutes of conversational setup later, you have a Claude Code that:
- Remembers you across sessions (tiered memory + automatic hygiene)

- Self-improves via a daily scheduled task that surfaces new Claude Code releases
- Snapshots its own memory defensively (PreCompact + SessionEnd hooks)
- Has a coherent voice and behavior profile built up from real-world feedback
