Don't keep reiterating the task list with every response.
**Why:** The user can see the tasks (they're rendered in the UI). Repeating them in prose adds tokens and friction.
**How to apply:** Only reference tasks when directly relevant (completing one, creating one, or the user asks about status). Never dump the full list as part of a normal response.
